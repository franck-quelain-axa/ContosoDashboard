## New Entities

### 1. `Document`

Stores metadata for every uploaded file. File content lives on the filesystem; only the relative path is stored here.

```csharp
// Models/Document.cs
public class Document
{
    [Key]
    public int DocumentId { get; set; }              // integer PK — consistent with User, Project

    [Required, MaxLength(255)]
    public string Title { get; set; } = string.Empty;

    [MaxLength(2000)]
    public string? Description { get; set; }

    [Required, MaxLength(100)]
    public string Category { get; set; } = string.Empty;  // text value, not enum int (per spec)
    // Allowed values: "Project Documents", "Team Resources", "Personal Files",
    //                 "Reports", "Presentations", "Other"

    [MaxLength(1000)]
    public string? Tags { get; set; }                // comma-separated free-text tags

    [Required]
    public int UploadedByUserId { get; set; }

    public int? ProjectId { get; set; }              // null = personal document

    public int? TaskId { get; set; }                 // null = not attached to a specific task

    [Required, MaxLength(500)]
    public string FilePath { get; set; } = string.Empty;
    // Relative path pattern: {userId}/{projectId-or-"personal"}/{guid}.{ext}
    // Never user-supplied; always GUID-based

    [Required, MaxLength(255)]
    public string OriginalFileName { get; set; } = string.Empty;
    // Stored for display only; never used in filesystem paths

    [Required, MaxLength(255)]
    public string FileType { get; set; } = string.Empty;
    // MIME type, e.g. "application/vnd.openxmlformats-officedocument.wordprocessingml.document"
    // 255 chars to accommodate long Office MIME types (per spec constraint)

    [Required]
    public long FileSize { get; set; }               // bytes

    public DateTime UploadedDate { get; set; } = DateTime.UtcNow;

    // Navigation properties
    [ForeignKey("UploadedByUserId")]
    public virtual User UploadedBy { get; set; } = null!;

    [ForeignKey("ProjectId")]
    public virtual Project? Project { get; set; }

    [ForeignKey("TaskId")]
    public virtual TaskItem? Task { get; set; }

    public virtual ICollection<DocumentShare> Shares { get; set; } = new List<DocumentShare>();
    public virtual ICollection<ActivityLog> ActivityLogs { get; set; } = new List<ActivityLog>();
}

public static class DocumentCategories
{
    public const string ProjectDocuments = "Project Documents";
    public const string TeamResources    = "Team Resources";
    public const string PersonalFiles    = "Personal Files";
    public const string Reports          = "Reports";
    public const string Presentations    = "Presentations";
    public const string Other            = "Other";

    public static readonly string[] All =
    {
        ProjectDocuments, TeamResources, PersonalFiles, Reports, Presentations, Other
    };
}
```

**Validation rules** (enforced in `DocumentService`):
- `Title`: required, 1–255 characters
- `Category`: must be one of `DocumentCategories.All`
- `FileSize`: > 0 and ≤ 26,214,400 bytes (25 MB)
- `FileType` / extension: must be in the whitelist (see Contracts)
- `FilePath`: generated by service — never accepted from user input

### 2. `DocumentShare`

Records an explicit sharing grant from an owner to a recipient. A recipient is either a specific user or a project (all current members of that project).

```csharp
// Models/DocumentShare.cs
public class DocumentShare
{
    [Key]
    public int DocumentShareId { get; set; }

    [Required]
    public int DocumentId { get; set; }

    [Required]
    public int SharedByUserId { get; set; }

    [Required]
    public ShareRecipientType RecipientType { get; set; }
    // User  → RecipientId is a UserId
    // Project → RecipientId is a ProjectId (all current members inherit access)

    [Required]
    public int RecipientId { get; set; }

    public DateTime SharedDate { get; set; } = DateTime.UtcNow;

    // Navigation properties
    [ForeignKey("DocumentId")]
    public virtual Document Document { get; set; } = null!;

    [ForeignKey("SharedByUserId")]
    public virtual User SharedBy { get; set; } = null!;
}

public enum ShareRecipientType
{
    User,
    Project
}
```

**Access resolution** (evaluated at query time in `DocumentService.CanUserAccessDocumentAsync`):

A user `U` can access document `D` if **any** of the following is true:
1. `D.UploadedByUserId == U.UserId` (owner)
2. `DocumentShare` exists where `RecipientType == User && RecipientId == U.UserId`
3. `DocumentShare` exists where `RecipientType == Project && RecipientId IN (U's ProjectMember.ProjectIds)`
4. `D.ProjectId IN (U's ProjectMember.ProjectIds)` (project document visible to all members)
5. `U.Role == UserRole.Administrator` (full access per spec)
6. `U.Role == UserRole.TeamLead && D.UploadedByUserId IN (U's team member user IDs)` (read-only; see R-003)

### 3. `ActivityLog`

Immutable audit record of every document operation.

```csharp
// Models/ActivityLog.cs
public class ActivityLog
{
    [Key]
    public int ActivityLogId { get; set; }

    [Required]
    public int DocumentId { get; set; }

    [Required]
    public int ActorUserId { get; set; }

    [Required]
    public DocumentActivityType ActivityType { get; set; }

    public DateTime ActivityDate { get; set; } = DateTime.UtcNow;

    [MaxLength(500)]
    public string? Details { get; set; }             // optional context (e.g., shared-to username)

    // Navigation properties
    [ForeignKey("DocumentId")]
    public virtual Document Document { get; set; } = null!;

    [ForeignKey("ActorUserId")]
    public virtual User Actor { get; set; } = null!;
}

public enum DocumentActivityType
{
    Uploaded,
    Downloaded,
    Previewed,
    Deleted,
    MetadataEdited,
    FileReplaced,
    Shared,
    AttachedToTask
}
```

### New EF configuration in `OnModelCreating`

```csharp
// --- Document ---
modelBuilder.Entity<Document>()
    .HasOne(d => d.UploadedBy)
    .WithMany()
    .HasForeignKey(d => d.UploadedByUserId)
    .OnDelete(DeleteBehavior.Restrict);  // retain docs on user deactivation (FR-032)

modelBuilder.Entity<Document>()
    .HasOne(d => d.Project)
    .WithMany()
    .HasForeignKey(d => d.ProjectId)
    .OnDelete(DeleteBehavior.SetNull);   // documents survive project deletion

modelBuilder.Entity<Document>()
    .HasOne(d => d.Task)
    .WithMany()
    .HasForeignKey(d => d.TaskId)
    .OnDelete(DeleteBehavior.SetNull);

// Indexes — drive sorting/filtering performance (FR-008, FR-009, FR-010)
modelBuilder.Entity<Document>().HasIndex(d => d.UploadedByUserId);
modelBuilder.Entity<Document>().HasIndex(d => d.ProjectId);
modelBuilder.Entity<Document>().HasIndex(d => d.UploadedDate);
modelBuilder.Entity<Document>().HasIndex(d => d.Category);

// --- DocumentShare ---
modelBuilder.Entity<DocumentShare>()
    .HasOne(s => s.Document)
    .WithMany(d => d.Shares)
    .HasForeignKey(s => s.DocumentId)
    .OnDelete(DeleteBehavior.Cascade);  // shares deleted when document is deleted

modelBuilder.Entity<DocumentShare>()
    .HasIndex(s => new { s.DocumentId, s.RecipientType, s.RecipientId })
    .IsUnique();  // prevent duplicate share grants

// --- ActivityLog ---
modelBuilder.Entity<ActivityLog>()
    .HasOne(a => a.Document)
    .WithMany(d => d.ActivityLogs)
    .HasForeignKey(a => a.DocumentId)
    .OnDelete(DeleteBehavior.Cascade);

modelBuilder.Entity<ActivityLog>()
    .HasIndex(a => new { a.DocumentId, a.ActivityDate });
modelBuilder.Entity<ActivityLog>()
    .HasIndex(a => a.ActorUserId);
```

## State Transitions

### Document lifecycle

```
[file selected] → Validated → Scanned → Saved to disk → Metadata saved → Active
                                                                           ↓
                                                                   [owner deletes]
                                                                           ↓
                                                               File deleted + DB record deleted
                                                               ActivityLog: Deleted recorded
```

**Ownership transfer on account deactivation** (FR-032):
- `DocumentService.TransferOwnershipAsync(deactivatedUserId)` sets `UploadedByUserId` to the first active Administrator UserId via:  
  `SELECT TOP 1 UserId FROM Users WHERE Role = UserRole.Administrator ORDER BY UserId`
- Called by user management code (outside scope of this feature) when an account is deactivated.

## EF Core Migration

**Migration name**: `AddDocumentManagement`  
**Command**: `dotnet ef migrations add AddDocumentManagement --project ContosoDashboard`

Creates:
- `Documents` table
- `DocumentShares` table
- `ActivityLogs` table
- All indexes defined above

Does not alter any existing table. Safe to apply to a database with existing seeded data.

**Apply**: `dotnet ef database update --project ContosoDashboard`
